## 调研结果：
   
   	- 同一猫池3大运营商线不能混插，即至少需要3台电脑组装3套环境
   	- 猫池插槽经常会有 注册网络失败的问题，切换2g/3g/4g均不行，未发现固定规律，可能是硬件问题。待于厂商沟通
   	- 已和鸟妹、骨贝、魏紫 沟通，初步确定：
   		- 完全依靠猫池来组装系统，主要原因是支持实时获取短信及稳定性；
   		- 采购4G全网通猫池，主要是支持不插拔卡情况下造流量
   		- 短信获取可以通过API读取MySQL获取，其他如拨号、造流量 等需要依靠配套软件，无API可提供
